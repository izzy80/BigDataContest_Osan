# BigDataContest_Osan
>기간 : 20.12.15 ~ 21.01.28  
>인원 : 4명
>사용언어 : R, Python 
>역할 :   
>1. 과제 1 담당  
> - 데이터 수집 및 가공  
>    EDA를 통해 데이터에서 유의미한 파생변수 추출  
> - 데이터 분석  
>   로지스틱 회귀모델을 이용하여 데이터 분석 실시  
>   상관분석을 실시하여 다중공선성 여부를 파악  
>   모델의 설명력을 높이고자 단계적 변수선택법을 이용함   
>2. PPT 제작
